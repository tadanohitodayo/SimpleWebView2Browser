# M365 AI Explorer (Powered by WebView2 & Microsoft Graph)

## 🚀 Project Overview
This is a high-performance custom browser built with **.NET 10** and **Visual Studio 2026**.
My mission is to integrate **Microsoft Graph API** to create an AI-powered workspace that summarizes emails and calendars in real-time.

## 🛠 Tech Stack & Achievements
- **Browser Engine**: Microsoft WebView2
- **Auth**: MSAL (Microsoft Authentication Library) with `InteractiveBrowserCredential`
- **Graph SDK**: Updated to **Microsoft Graph SDK v5**
- **AI Features**: Real-time web page summarization logic implemented

## 🎯 Current Status & Goal
I have successfully implemented the authentication flow and Graph client. 
**I am currently waiting for Microsoft 365 E5 Sandbox approval** to perform end-to-end testing of Outlook and Calendar integrations.
